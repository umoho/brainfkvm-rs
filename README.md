# `BrainfkVM-rs`

## Descriptions

A simple implement for Brainfuck.
Once can only read one file.

## Usage

Very simply,

```bash
$ brainfkvm <FILEPATH>
```

and you can read help with

```bash
$ brainfkvm --help
```

Well, filepath cannot be `--help`... XD

## Build

Use Cargo,

```bash
$ cargo build
$ cargo run -- <FILEPATH or --help>
```
